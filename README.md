# Kodiflya

An Android app that makes algorithms easy to see and understand. Sorting, pathfinding, and tree traversals animate in real time so you can watch them think step by step.

## What this is

A study project exploring how to build Android apps with AI assistance — using Claude Code from Anthropic to design, implement, and review code through agents and skills.

The goal is not just the app itself, but learning how to collaborate with AI effectively throughout the full development cycle.

## Stack

Kotlin · Jetpack Compose · Hilt · MVVM · Material 3

## Algorithms

**Sorting** — Bubble, Insertion, Merge, Quick

**Pathfinding** — BFS, DFS, Dijkstra

**Trees** — BST Inorder, Preorder, Postorder
